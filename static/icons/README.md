# Aircraft Icons

SVG icons for different aircraft categories. All icons should be monochrome side-view silhouettes.

## Sources
- Download free SVG icons from https://www.svgrepo.com
- Search terms: "airplane", "aircraft", "jet", "helicopter", "propeller"

## Icons Needed
1. `narrowbody.svg` - Single aisle jets (A320, B737, E190)
2. `widebody.svg` - Twin aisle jets (A330, B777, B787)
3. `turboprop.svg` - Propeller aircraft (ATR, Q400)
4. `business.svg` - Business jets (CL60, GLEX)
5. `piston.svg` - Small GA aircraft (C172, PA28)
6. `helicopter.svg` - Helicopters
7. `unknown.svg` - Default fallback

## SVG Requirements
- Monochrome (use `fill="currentColor"`)
- Side view profile
- Simple, clean design
- No background
- 24x24 viewBox recommended
